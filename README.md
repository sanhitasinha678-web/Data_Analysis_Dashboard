# Customer-Behaviour-Analysis (Interactive Dashboard Creation using MS Excel)
This project focuses on analyzing customer behavior using Excel, with data transformed and structured using Power Query and insights visualized through an interactive dashboard. The dataset consists of 3,900 customers, and the objective was to understand customer demographics, purchasing patterns, and engagement levels.
## Key Insights
- Identified distribution of customers across loyalty segments.
- Observed different engagement levels based on purchase frequency.
- Highlighted customer preferences in payment methods and product categories.
- Evaluated subscription adoption among customers.
